# Video Transcribe

Автоматизация follow-up после звонков.

## Проблема

- 4-5 часов звонков в день
- Позиция: директор инженерии
- ~10 параллельных проектов
- Разные контексты звонков
- Нужно систематизировать фоллоу-апы

## Текущий процесс

```
Google Meet / Zoom → OBS → локальный видеофайл → ???
```

## Решение

CLI-скрипт для автоматической транскрибации и генерации summary встреч.

## Пайплайн

```
Видео → Аудио → Нарезка → Транскрибация (с диаризацией) →
Склейка → Наложение контекста → Markdown с follow-up
```

## Входные данные

1. Путь к видеофайлу (записан через OBS)
2. Промпт с контекстом встречи (свободная форма)
3. Куда сохранить результат

## Выход (Markdown)

- **Summary встречи** — краткое содержание
- **Задачи по спикерам** — кто за что отвечает
- **Принятые решения** — кто согласен/не согласен
- **Интересные мысли** — инсайты и идеи

## Язык

Русский с IT-контекстом (англицизмы: follow-up, env, ресерч и т.п.)
